git-upstream(1) -- Add/update remote upstream repository
================================

## SYNOPSIS

`git-upstream` &lt;url&gt;

## DESCRIPTION

  Adds or updates the remote upstream url.

## OPTIONS

  &lt;url&gt;

  Remote url.

## EXAMPLES
   
  $ git upstream git://github.com/visionmedia/git-extras.git
